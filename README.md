### Instructions

1. Clone the repo
   ```sh
   git clone https://github.com/lucaspineda/music-platform.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the node server
   ```sh
   node server.js
   ```
4. Start the Next server
   ```sh
   npm run dev
   ```
5. Open `https://localhost:3000`
